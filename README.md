# NFC Tag Maestro

A beautifully designed Android NFC tag manager. Read, write, clone, lock, and erase NFC tags.

## Features

- **Scan** — UID, type, memory, ATQA/SAK, manufacturer, NDEF records (tappable URLs)
- **Write** — URLs, text, Wi-Fi, phone, email, SMS with input validation
- **Lock** — Permanently make tags read-only (irreversible)
- **Clone** — Batch duplicate with per-tag logging
- **Erase** — Format/wipe with visual tap prompt
- **History** — Swipe to delete, clear all, tappable URL entries
- **About** — Built-in help with links
